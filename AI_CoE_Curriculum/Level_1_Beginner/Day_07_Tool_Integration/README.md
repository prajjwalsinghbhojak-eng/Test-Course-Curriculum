# Day 7 — Tool Integration & Introduction to Agentic AI

## Learning Objectives
- Conceptually understand how Language Models can be extended using external Tools (Function Calling).
- Implement standard function calling workflows with the Gemini API to retrieve real-time or exact logic responses.
- Move from basic tools to Agentic AI logic: letting models autonomously decide when and how to call tools.
- Gain familiarity with lightweight agent frameworks, such as `smolagents`.

## Prerequisites
- Deep understanding of standard LLM capabilities and limitations (from Days 1-3).
- Knowledge of basic Python functions.


## Required Reading
- [Function Calling and Tool Use Introduction](https://platform.openai.com/docs/guides/function-calling)
