# Day 3 — Advanced Prompt Engineering & Structured Output

## Learning Objectives
- Apply advanced prompting: self-consistency, tree-of-thought, iterative refinement
- Generate structured outputs (JSON mode, schema enforcement, Pydantic + Gemini)
- Understand the ReAct prompting pattern (Reason + Act)
- Build a reusable prompt library for common business tasks

## Prerequisites
- Completion of Day 2 (Prompt Engineering Fundamentals)

## Required Reading
- [ReAct: Synergizing Reasoning and Acting in Language Models](https://arxiv.org/abs/2210.03629)
- [Gemini Structured Output Docs](https://ai.google.dev/gemini-api/docs/structured-output)

## Video Lectures
