# Day 4 — Hallucinations, Grounding & Generation Control

## Learning Objectives
- Understand why LLMs hallucinate and categorize hallucination types
- Apply grounding techniques to reduce hallucinations
- Control generation with temperature, top-k, top-p, and safety settings
- Implement input validation and output sanitization patterns

## Prerequisites
- Completion of Day 3 (Advanced Prompt Engineering)

## Required Reading
- [Grounding with Google Search in Gemini](https://ai.google.dev/gemini-api/docs/grounding)
