# Day 4 — Retrieval Augmented Generation (RAG)

## Learning Objectives
- Understand the fundamentals of Retrieval Augmented Generation (RAG) and Semantic Search.
- Learn to build RAG applications using the Gemini managed File Search API.
- Grasp how Vector Databases (such as Qdrant) integrate with Gemini Models for similarity search.

## Prerequisites
- Basic Python and API knowledge.
- Understanding of embeddings (from Day 2).
