# AI Center of Excellence (CoE) - L&D

## Objectives
- Train team members in AI/ML fundamentals.

- Standardize Python 3.12 development practices within the CoE.

- **Focus:** Generative AI (LLMs, Prompt Engineering, RAG, AI Assistants, Gen AI Applications)
