LLM Evaluation Recipes:

* This collection focuses on evaluating LLM and RAG systems
* Covers automated evaluation, LLM-as-a-judge, and benchmarking techniques
* Each notebook is independent and can be run separately

---

Notebooks Overview:

* RAG Evaluation
  File: rag_evaluation.ipynb
  Focus:

  * Evaluating RAG pipelines
  * Metrics (relevance, faithfulness)
  * Benchmarking retrieval + generation quality

* Using LLM-as-a-judge for an automated and versatile evaluation
  File: llm_judge.ipynb
  Focus:

  * LLM-based evaluation
  * Automated scoring of responses
  * Replacing manual evaluation

* Evaluating AI Search Engines with `judges` - the open-source library for LLM-as-a-judge evaluators
  File: llm_judge_evaluating_ai_search_engines_with_judges_library.ipynb
  Focus:

  * Advanced LLM-as-a-judge pipelines
  * Using the `judges` library
  * Evaluating full search systems

Notes:
