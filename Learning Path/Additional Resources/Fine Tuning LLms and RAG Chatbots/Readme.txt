LLM Fine-Tuning Recipes:

* This collection focuses on fine-tuning techniques for LLMs
* Covers prompt tuning, PEFT, synthetic data, and function calling
* Each notebook is independent and can be run separately

---

Notebooks Overview:

* Prompt tuning with PEFT
  File: prompt_tuning_peft.ipynb
  Focus:

  * Prompt tuning basics
  * Parameter-Efficient Fine-Tuning (PEFT)
  * Lightweight adaptation of LLMs

* Fine-tuning T5 for Automatic GitHub Tag Generation with PEFT
  File: finetune_t5_for_search_tag_generation.ipynb
  Focus:

  * Task-specific fine-tuning
  * Tag generation use case
  * Applying PEFT on encoder-decoder models

* Documentation Chatbot with Meta Synthetic Data Kit
  File: fine_tune_chatbot_docs_synthetic.ipynb
  Focus:

  * Synthetic data generation
  * Chatbot fine-tuning
  * Improving domain-specific responses

* Fine-tuning a Code LLM on Custom Code on a single GPU
  File: fine_tuning_code_llm_on_single_gpu.ipynb
  Focus:

  * Code model fine-tuning
  * Running on limited hardware (single GPU)
  * Efficient training strategies

* Fine-tuning LLMs for Function Calling with the xLAM Dataset
  File: function_calling_fine_tuning_llms_on_xlam.ipynb
  Focus:

  * Function/tool calling fine-tuning
  * Structured outputs
  * Advanced LLM capabilitiess

Notes:
